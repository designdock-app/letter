# saythis.lol clone

Static clone of https://www.saythis.lol/ captured from the live production build.

## Files

- `index.html` - entry HTML document
- `assets/index-Ds9mZN1O.js` - compiled application bundle
- `assets/index-DRrQzLth.css` - compiled stylesheet
- `assets/pigeon-JPt7TM73.png` - image asset used by the app
- `favicon.ico` and `og-image.png` - site metadata assets

## Run locally

Serve the repository with any static server from the repo root, for example:

```bash
python3 -m http.server 8080
```

Then open `http://localhost:8080`.
